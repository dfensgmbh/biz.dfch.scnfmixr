# Links to sources

Below you find links to the sources of the software that is necessary for scnfmixr to operate.

Some libraries and Raspberry Pi OS with Linux Debian Trixie are not shown.

https://github.com/kmatheussen/jack_capture

## zita-ajbridge
https://kokkinizita.linuxaudio.org/linuxaudio/

## JACK2 (jack2 server / jackd2)
https://github.com/jackaudio/jack2

https://github.com/MusicPlayerDaemon/MPD

https://github.com/MusicPlayerDaemon/mpc
